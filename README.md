Mô tả ứng dụng: Xây dựng ứng dụng đặt phòng khách sạn đơn giản

Hướng dẫn cài đặt: 
- npm install && npm run build
- Backend: npm run start:dev
- Frontend: npm run dev
  
API endpoint: 
- GET /bookings
- GET /bookings/:id
- POST /bookings
- DELETE /bookings/:id
- GET /rooms/:id
- GET /rooms?location=&checkin=&checkout=
